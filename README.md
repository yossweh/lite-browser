# LiteBrowser 🌐

A lightweight, fast browser for Android. Under 2MB!

## Features

- 🚀 **Lightweight** - Under 2MB APK size
- 🛡️ **Built-in Ad Blocker** - Block annoying ads and trackers
- 🌙 **Dark Mode** - Easy on your eyes
- 📑 **Tab Management** - Multiple tabs support
- 🔍 **Multiple Search Engines** - Google, Bing, DuckDuckGo
- 📱 **User Agent Switching** - Desktop/Mobile mode
- ⬇️ **Download Manager** - Download files easily
- 🔖 **Bookmarks** - Save your favorite sites
- 📜 **History** - Browse with confidence
- 🔒 **Privacy Focused** - No tracking, no data collection

## Download

Download the latest APK from [Releases](../../releases).

## Build from Source

1. Clone the repository:
```bash
git clone https://github.com/yossweh/lite-browser.git
cd lite-browser
```

2. Build the APK:
```bash
./gradlew assembleDebug
```

3. The APK will be at:
```
app/build/outputs/apk/debug/app-debug.apk
```

## Permissions

- `INTERNET` - Access the internet
- `ACCESS_NETWORK_STATE` - Check network status
- `WRITE_EXTERNAL_STORAGE` - Save downloads
- `READ_EXTERNAL_STORAGE` - Read downloaded files
- `CAMERA` - QR code scanning
- `ACCESS_FINE_LOCATION` - Location-based services

## Privacy

LiteBrowser does NOT collect any personal data. All data stays on your device.
